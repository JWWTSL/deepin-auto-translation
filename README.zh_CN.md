# deepin-auto-translation

在翻译者参与贡献之前，使用 LLM 预先填充 Qt Linguist TS 文件中未翻译的字符串。

预先填充的字符串会被标记为 `unfinished`，以此让翻译者可以得知这些字符串需要被主动校对。
